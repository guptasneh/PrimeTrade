# Hyperliquid Trader Behavior vs Market Sentiment

## Objective
Analyze how Bitcoin market sentiment (Fear vs Greed) influences trader behavior and performance on Hyperliquid, and derive actionable trading rules.

---

## Datasets
1. Bitcoin Fear & Greed Index  
2. Historical trader data
